Research

During the last four semesters at Park University's Honors Academy, my research has been to combine current technologies in order to build an augmented reality guidance system
for firefighters. This guidance system will help firefighters find both victims and egress points in low and high visibility situations. This research includes using a Google Glass device
as the heads-up-display (HUD). The mobile application provides the augmented view which filters out the fire and heavy smoke of a building engulfed in fire. The application built during
this research uses both Java and Android technologies to augment a real-world view by using the Sobel edge-detection method.
